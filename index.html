HOW TO

Install in 3 steps

1
Open Kettu Settings
Tap your profile → Settings → scroll to Kettu → Plugins
2
Tap + and paste URL
Tap the blue + button, paste plugin URL:
https://tanhoangviet.github.io/BetterCodeBlocks/BetterCodeBlocks
3
Enable & enjoy
Toggle the plugin on. Code blocks will be automatically highlighted!
