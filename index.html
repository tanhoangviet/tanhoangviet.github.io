// how to install

Install in 3 steps

1
Open Kettu Settings
Tap your profile → Settings → scroll down to Kettu → Plugins
2
Tap + and paste the URL
Tap the blue + button, then paste the plugin URL:
https://tanhoangviet.github.io/plugins/BetterCodeBlocks
3
Tap Install & enjoy
Kettu will fetch the manifest and install the plugin automatically. No restart needed!
